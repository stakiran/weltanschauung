About the Three Labels

Adoption Scope

LevelDescription
3Company-wide (provided to all employees)
2Project or Group (provided to a specific project or organizational unit)
1Individual (used by specific individuals)

Approval Requirement

  • No approval required: Can be used without any formal request, or requests are open and essentially anyone can get it.
  • Approval required: Requires managerial approval or submission of application documents to the administration.

Proxy Availability

  • Direct access: Original tool can be accessed directly (e.g., a developer can call the OpenAI API directly).
  • Proxy required: Access through a proxy service (e.g., a wrapper service for the OpenAI API).
