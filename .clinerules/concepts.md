# 概念

## ヴェルタンシャウン
**ヴェルタンシャウン(weltanschauung)** とは、ツールの世界観を指す。

ヴェルタンシャウンは言語による端的な説明や概念から成る。これらを読むことで、読者はそのツールがどんな世界観を持つかを想像する。

## ヴェルドック
**ヴェルドック(weltanschauung documents)** とは、ヴェルタンシャウンを表現した文書を指す。

ヴェルドックには、あるツールを表現するための端的な説明や概念の解説が書いてある。

ヴェルドックはデザインドキュメントやその他設計書というほどテクニカルな情報は含まない。あくまでもツールの根幹を成す（世界観の構成要素となる）概念や最低限の説明を出すのみ。

## プロトタイピング
ヴェルタンシャウンの文脈において、**プロトタイプ** とはヴェルドックをもとにしてモックをつくることである。

モックは一枚の HTML から構成される。JS や CSS も HTML の中に含める。これにより、利用者は HTML ファイルをブラウザで開くことで、そのヴェルタンシャウンをプロトタイプとして体験できる。無論、一枚の HTML にすぎないので、ウェブアプリほど潤沢な体験はできない。あくまでもプロトタイプにすぎない。
